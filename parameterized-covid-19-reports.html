3 Parameterized Covid-19 reports

Below are the results from running a parameterized Rmarkdown file with 3 different parameter environments.

configs <- list(
  list(country = "Netherlands", year = "2021", month = "8"),
  list(country = "Bulgaria", year = "2022", month = "7"),
  list(country = "Estonia", year = "2020", month = "9")
)

for (i in seq_along(configs)) {
  outfile <- paste0("data/media/report_", i, ".html")
  rmarkdown::render(
    input = "_covid-19_graphing.Rmd",
    params = configs[[i]],
    output_file = outfile,
    quiet = TRUE
  )
  cat(paste0('<iframe src="', outfile, '" width="100%" height="600px"></iframe>\n'))
}
